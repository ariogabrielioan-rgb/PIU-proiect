Tema aplicatie practica:
Aplicatie de tip examen grila cu urmatoarele butoane grafice:
Sustine Examen si Realizare Examen
Butonul Sustine duce la o interfata unde se introduc datele studentului care sustine examenul(nume,prenume,id,grupa/anul)
Studentul alege materia si profesorul de la examen prin butoane drop-down menu
Butonul Realizare Examen duce la o interfata unde se regasesc butoane :selecteaza materia si numele profesorului 
Dupa selectare se pot adauga intrebarile grila intr un text box care verifica daca s a scris ceva,daca nu,se finalizeaza realizarea examenului cu succes.
Detalii suplimentare/design:
Sa se afiseze nr de raspunsuri corecte la final
Sa se afiseze mesajul cu picat/respins daca nota este sub un anumit prag
sa se afiseze mesajul felicitari daca nota este peste 9,5
(discutabil cu notele)
(design frumos cu flori sau minimalist ca sa nu se sperie studentii)
